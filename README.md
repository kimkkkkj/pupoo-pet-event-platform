# Pupoo

반려동물 박람회·페스티벌을 탐색하고 신청·결제까지 하는 행사 플랫폼.
사용자 사이트와 운영자용 관리자 콘솔로 나뉘며, 실시간 혼잡도·안내 챗봇·
포스터 자동 생성·혼잡도 예측 같은 AI 기능을 붙였다.

팀 프로젝트이며, 프론트엔드·UI/UX를 단독 담당했다.
사용자 사이트와 관리자 콘솔 전 화면의 설계·구현, 디자인 시스템, 반응형을 맡았고,
담당 영역의 DB 연동, AI 기능(챗봇·포스터) 연동, 그리고 관리자에서 등록·변경한
내용이 홈(사용자) 화면에 바로 반영되도록 하는 연동까지 작업했다.
(백엔드 코어·AI 서버·인프라는 팀원들이 담당.)

## 기술 스택
- Frontend — React, Vite, React Router
- DB — MySQL 8 (담당 영역 스키마·쿼리)
- AI — 챗봇·포스터 생성 등 AI 기능 프론트 연동

## 주요 기능
**사용자** — 행사 탐색·신청, KakaoPay 결제/환불, 프로그램 신청·QR 입장,
커뮤니티(후기·갤러리·Q&A), 실시간 혼잡도·대기 현황, 안내 챗봇 '푸리'.

**관리자** — 행사·참가자·결제 관리, 커뮤니티/신고 모더레이션, 통계 대시보드,
운영 보조 챗봇 '누리', AI 포스터 생성기, 혼잡도 예측 시각화.
관리자에서 등록한 행사 정보가 홈 화면에 즉시 반영되도록 연동.
